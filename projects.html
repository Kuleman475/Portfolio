Whats My Weather

Languages Used: HTML, CSS, JavaScript

Whats my weather website is a website that you can see the weather for 3 towns, Preston ID, Fish Haven ID, and Soda Springs ID. It shows the weather summary which contains temperature, wind chill (if applicable), wind speed, humidity, and current weather outside (partly cloudy, sunny, etc.). Also shows the 5 day forecast. There is a page on the website in which you can report a storm in those areas which contain what type of storm it was, when it occured, wheter you were safe, etc.
